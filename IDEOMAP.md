# IDEOMAP - Mapa de Conocimiento y Expansión Ideológica

Este documento sirve como guía para la expansión del contenido intelectual y las ideas presentadas en la web "Nueva España".

## 1. Núcleo Ideológico (Fundamentos)
- [ ] **Separación de Poderes**: Explicación clara de origen vs. ejercicio.
- [ ] **Representación**: Mandato imperativo vs. representativo.
- [ ] **Partitocracia**: Análisis del sistema actual y sus fallos.

## 2. Estrategia de Comunicación
- [ ] **Storytelling Visual**: Cómo traducir conceptos abstractos (libertad política) en gráficos.
- [ ] **Glosario de Términos**: Definiciones precisas para evitar la neolengua (ej. "Democracia", "Estado de Derecho").
- [ ] **Biblioteca Digital**: Recopilación de textos y referencias (Trevijano, Arendt, etc.).

## 3. Movilización (La Acción)
- [ ] **Abstención Activa**: Explicación estratégica y moral.
- [ ] **Asociacionismo Civil**: Guía para crear tejido civil al margen del estado.
- [ ] **Resistencia Pacífica**: Métodos y ejemplos históricos.

## 4. Datos y Evidencia (La Realidad)
- [ ] **Mapa de Corrupción**: Base de datos de casos por región y partido.
- [ ] **Deuda Pública**: Visualización del endeudamiento real.
- [ ] **Control de Medios**: Gráficos de subvenciones a medios de comunicación.
